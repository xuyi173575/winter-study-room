# 冬日书房 - 快速开始

## 🚀 5分钟快速启动

### 第一步：安装依赖
```bash
npm install
```

### 第二步：启动开发服务器
```bash
npm run dev
```

### 第三步：打开浏览器
访问显示的本地地址（通常是 http://localhost:5173）

## 🎯 立即体验

1. **点击"点燃篝火"按钮** - 开始你的冬日书房之旅
2. **调节音量** - 使用底部控制面板的滑块
3. **探索彩蛋** - 点击场景中的各个元素

## 📝 主要功能

### 环境音控制
- 🔥 篝火声（默认70%）
- 🌧️ 雨声（默认30%）
- 📖 翻书声（默认20%）
- 🌊 白噪音（默认40%）

### 互动彩蛋
- 📚 点击书本 → 查看名言
- 🍓 点击草莓书（第4本红色书）→ 圣诞祝福
- 🪟 点击窗户 → 雨声增加20%
- ☕ 点击咖啡杯 → 查看名言
- 🦀 点击螃蟹 → 看它横着走

### 专注定时器
- 设置1-180分钟
- 快捷按钮：15/25/45/60分钟
- 倒计时结束自动停止音效

## 🎨 推荐配置

### 修图时
```
篝火: 70%
白噪音: 40%
定时器: 60分钟
```

### 阅读时
```
篝火: 50%
翻书: 30%
雨声: 20%
```

### 工作学习
```
白噪音: 60%
定时器: 25分钟（番茄钟）
```

### 睡前放松
```
雨声: 50%
白噪音: 40%
```

## 🔧 音频文件配置（可选）

当前使用占位符音频。如需替换：

1. 准备音频文件（MP3格式）
2. 放入 `public/audio/` 目录
3. 修改 `src/hooks/useAudioManager.ts`

## 📦 构建生产版本

```bash
npm run build
```

构建后的文件在 `dist` 目录，可直接部署。

## ❓ 常见问题

**Q: 为什么听不到声音？**
A: 首次播放需要用户交互。点击"点燃篝火"后，调节音量滑块即可。

**Q: 设置会保存吗？**
A: 会的！音量设置自动保存在浏览器本地。

**Q: 支持手机吗？**
A: 支持！响应式设计，手机和电脑都能完美使用。
