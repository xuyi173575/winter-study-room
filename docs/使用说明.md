# 冬日书房 - 使用说明

## 🎁 这是什么？

这是一个专为你设计的温暖治愈的互动网页应用。每次你需要专注工作、阅读或放松时，都可以打开它，让温暖的氛围音陪伴你。

## 🌟 特色功能

### 1. 点燃篝火仪式
- 打开网页后，你会看到一个静谧的书房场景
- 点击"点燃篝火"按钮，火焰会从小到大慢慢点燃
- 3秒后，控制面板会从底部优雅地出现

### 2. 环境音控制
你可以独立调节4种环境音的音量：

| 音效 | 默认音量 | 适合场景 |
|------|----------|----------|
| 🔥 篝火 | 70% | 营造温暖氛围 |
| 🌧️ 雨声 | 30% | 增加安全感 |
| 📖 翻书 | 20% | 阅读时的真实感 |
| 🌊 白噪音 | 40% | 帮助专注 |

**小技巧**：
- 修图时：建议开启篝火70% + 白噪音40%
- 阅读时：建议开启篝火50% + 翻书30% + 雨声20%
- 冥想时：建议开启雨声60% + 白噪音30%
- 助眠时：建议开启雨声50% + 白噪音40%

### 3. 专注定时器（番茄钟）
- 设置1-180分钟的专注时长
- 倒计时结束后会自动停止所有音效
- 快捷按钮：15分钟、25分钟、45分钟、60分钟

### 4. 互动彩蛋

#### 📚 书架上的书本
- 点击任意书本，会随机显示一条名言
- 共有6条精心挑选的名言，关于摄影、生活、温暖

#### 🍓 草莓书（第4本红色书）
- 这是最特别的一本书
- 点击它会显示专属的圣诞祝福
- 这是我想对你说的话

#### 🪟 窗户
- 点击窗户，雨声会自动增加20%
- 窗外飘着雪花，营造冬日氛围

#### ☕ 咖啡杯
- 点击咖啡杯，也会显示随机名言
- 注意看杯子上方的蒸汽动画

#### 🦀 小螃蟹
- 点击螃蟹，它会横着走一段距离
- 就像名言里说的："有些路需要换个方向"

### 5. 自动保存
- 你的音量设置会自动保存
- 下次打开时，会恢复上次的设置
- 无需登录，数据保存在本地

## 💡 使用建议

### 修图时
1. 点燃篝火
2. 篝火音量调到70%
3. 白噪音调到40%
4. 设置60分钟定时器
5. 专注修图，让温暖的声音陪伴你

### 阅读时
1. 点燃篝火
2. 篝火50% + 翻书30% + 雨声20%
3. 点击书本，看看今天的名言
4. 开始阅读

### 工作学习时
1. 点燃篝火
2. 白噪音调到60%
3. 设置25分钟番茄钟
4. 专注工作

### 睡前放松
1. 点燃篝火
2. 雨声50% + 白噪音40%
3. 关掉其他音效
4. 闭上眼睛，放松

## 🎨 设计细节

### 配色
- 米白色到奶油色的温暖渐变背景
- 棕色系的木质元素（书架、壁炉）
- 草莓红的强调色（你最喜欢的）
- 冬日蓝的窗外天空

### 字体
- 标题使用 Cormorant Garamond（优雅的衬线体）
- 正文使用 Crimson Pro（易读的衬线体）
- 避免了常见的 Inter、Roboto 等字体

### 动画
- 雪花飘落（窗外）
- 火焰跳动（壁炉）
- 蒸汽上升（咖啡杯）
- 螃蟹走动（点击触发）

## 🎯 个性化元素

这个应用里藏着很多关于你的小细节：

1. **草莓** - 你最喜欢的水果，藏在红色的书里
2. **螃蟹** - 你养的小螃蟹，可以点击互动
3. **书架** - 因为你喜欢阅读
4. **摄影相关的名言** - 因为你是摄影师
5. **温暖的配色** - 符合你的审美

## 📱 浏览器支持

- Chrome / Edge（推荐）
- Firefox
- Safari

## ⚠️ 注意事项

1. **音频文件**：当前使用的是占位符音频，实际部署时需要替换为真实的环境音文件
2. **音量控制**：首次播放时，浏览器可能需要用户交互才能播放音频
3. **数据保存**：使用 localStorage，清除浏览器数据会丢失设置
