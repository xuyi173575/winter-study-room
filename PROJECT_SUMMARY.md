## 📝 重要说明

### 音频文件
当前使用占位符音频URL。实际部署时需要：

1. 准备4个高质量MP3文件：
   - fire.mp3（篝火声）
   - rain.mp3（雨声）
   - book.mp3（翻书声）
   - noise.mp3（白噪音）

2. 放入 `public/audio/` 目录

3. 修改 `src/hooks/useAudioManager.ts` 中的 `AUDIO_URLS` 配置

### 部署建议
- **推荐平台**: Vercel（零配置，免费）
- **备选平台**: Netlify、GitHub Pages
- **构建命令**: `npm run build`
- **输出目录**: `dist`
